#### Usage:

- `yarn add arxoft/pull`
- `ln -s node_modules/.bin/arxoftpull pull`
- `nano pull` and update the URL and database details (if the project is not yet the supported platforms)
- `./pull`

#### Supported platforms fot Database dumps:

- WordPress
